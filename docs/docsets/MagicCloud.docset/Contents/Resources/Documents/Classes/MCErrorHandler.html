MCErrorHandler

class MCErrorHandler<R: MCMirrorAbstraction>: Operation, MCRetrier

This class handles the various possible CKErrors in a uniform and thorough way. Must create a new instance for each error resolution to ensure property ‘executableBlock’ doesn’t get overwritten before func ‘executeBlock’ is called.

If non-standard error handling is required for a given situation, implementation should handle unique aspect of error handling in the completion block first, then all other errors should be passed to this system.

Operation requires local cache to conform to Recordable protocol for version conflict resolution.
